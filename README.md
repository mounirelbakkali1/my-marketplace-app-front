# PetsPlaza (End-User)

PetsPlza is single page Marketplace application using vue js (fully responsive) 



## Installation 

```sh
npm install
```

## Compile and Hot-Reload for Development

```sh
npm run dev
```

## Compile and Minify for Production

```sh
npm run build
```

## Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```
